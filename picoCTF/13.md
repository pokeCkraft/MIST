## 13
Cryptography can be easy, do you know what ROT13 is? 
cvpbPGS{abg_gbb_onq_bs_n_ceboyrz}

### Solution
the same thing as previous one, we basically want to add 13 to each letter, the idea is that it needs to wrap around.
like P(16) + 13 --> 29 --> 3 --> C

```
def rot13(text):
    ans = ""

    for x in text:
        if x.isalpha():
            if x.isupper():
                norm = 65
            else:
                norm = 97
#first convert to ascii, then make the number to mod26 then rot 13 then mod 26 again then convert back to proper ascii
            ans += chr((ord(x) - norm + 13) % 26 + norm)        
        else:                                                   
            ans += x    
    return ans

enc_flag =  "cvpbPGS{abg_gbb_onq_bs_n_ceboyrz}"

flag = rot13(enc_flag)


print(flag)

```
